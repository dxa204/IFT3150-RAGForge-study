# Références

- **Documentation Chonkie**
  Documentation officielle de la bibliothèque Python utilisée pour les algorithmes de segmentation (Semantic, Late Chunking, etc.).
  <https://docs.chonkie.ai/>

- **Project Gutenberg**
  Bibliothèque numérique de livres libres de droits, utilisée comme source de données textuelles pour les tests d'ingestion.
  <https://www.gutenberg.org/ebooks/219>
- **Documentation ChainForge**
  Guide de référence officiel pour la plateforme de "prompt engineering" visuelle sur laquelle est basé ce projet.
  <https://chainforge.ai/docs/>

- **Stopwords JSON (GitHub)**
  Dépôt contenant les listes de mots vides (stopwords) utilisées pour le nettoyage et le pré-traitement des textes.
  <https://gist.github.com/sebleier/554280?permalink_comment_id=3744516#gistcomment-3744516>
